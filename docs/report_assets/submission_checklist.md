# Submission Checklist

## Code Package

- Confirm `src/`, `tests/`, `scripts/`, and `docs/` are included.
- Confirm the repository runs from a clean environment after installing `requirements.txt`.
- Confirm `python -m pytest -q` passes.
- Confirm `python -m src.phase1_demo` runs.
- Confirm `python -m src.main` runs.

## README

- Confirm setup steps are correct.
- Confirm all run commands are correct.
- Confirm expected output descriptions match the actual demo output.

## Final Report

- Confirm the report is written in English.
- Confirm the report is within the page limit required by the specification.
- Confirm architecture, implementation, testing, conclusion, references, and team contributions are included.
- Confirm screenshots match the final code and outputs.

## Video

- Confirm the video length is within 5 minutes.
- Confirm the format is MP4, 16:9, and 720p.
- Confirm the video demonstrates Phase I and Phase II features.
- Confirm the video clearly shows tamper detection.

## Final Review

- Confirm member names and contributions are accurate.
- Confirm there is no placeholder text left in the report.
- Confirm file names and packaging format match the submission portal requirements.
